A secure, full-stack **e-commerce application** built with **Spring Boot**, **REST APIs**, **JWT authentication**, and **MySQL**, designed for robust product, order, and user management with role-based access control.

## Features

- **CRUD operations** for products, users, orders, and carts  
- **Role-based authorization** (Admin: manage products - add, update, delete ; Users: manage their carts and orders)
- **JWT authentication** for secure access  
- **Cart management**: add, update, clear, and checkout operations for authenticated users  
- **DTOs for structured data transfer**, ensuring clean separation between layers  
- **API tested with Postman** for reliability and data integrity  
- Currently integrating **React front-end** for a seamless full-stack experience  

## Tech Stack

- **Backend:** Spring Boot, Java, Spring Security (JWT), MySQL  
- **Tools:** Postman, Maven, Git/GitHub  
- **Front-end (in progress):** React
